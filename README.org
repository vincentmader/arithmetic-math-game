_Prerequisites_:
- As of yet, an installation of Rust & Cargo is required to play.

_Usage_:
- To run the game, navigate to the folder containing this file in your shell, then execute `make`.
- To display statistics about your performance, execute `make stats`

_Minor Problems_:
- For questions like $x^0=...$, the program will show `false` when answering correctly with `1`.
